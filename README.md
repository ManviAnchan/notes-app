# 📝 Notes App with Markdown Support

A clean, minimal notes application built with vanilla HTML, CSS, and JavaScript.
Write notes in Markdown and see them rendered live!

## 🌐 Live Demo
[Click here to open the app](https://ManviAnchan.github.io/notes-app)

## ✨ Features
- Create, edit, and delete notes
- Write in Markdown with live preview
- Notes auto-save as you type
- Data persists using localStorage
- Clean, responsive UI

## 🛠️ Built With
- HTML5 (Semantic markup)
- CSS3 (Flexbox, CSS Variables)
- JavaScript (DOM, localStorage, Events)
- [marked.js](https://marked.js.org/) (Markdown parsing)

## 🚀 How to Run Locally
1. Clone the repository
   git clone https://github.com/YOUR_USERNAME/notes-app.git

2. Open the folder
   cd notes-app

3. Open index.html with Live Server in VS Code
